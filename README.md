# Simple NFT Drop Contract

A lightweight and secure Ethereum-based smart contract for deploying NFT collections. This repository provides a single-file solution for creators looking to launch digital art on EVM-compatible chains like Ethereum, Polygon, or Arbitrum.

## Features
* **Standard Compliance:** Fully ERC-721 compliant via OpenZeppelin.
* **Supply Control:** Set a maximum supply to ensure scarcity.
* **Public Minting:** Simple function for users to mint tokens.
* **Owner Controls:** Easy-to-use functions to withdraw funds and update metadata.

## Deployment Guide
1. Open [Remix IDE](https://remix.ethereum.org/).
2. Create a file named `NFTDrop.sol`.
3. Copy the code from `NFTDrop.sol` in this repo.
4. Compile and deploy using Injected Provider (MetaMask).
